    Under the hood

    How the copilot decides.

    Navigation agent: directs fans to gates, concessions, toilets, and sections.

    Crowd agent: uses live sensor telemetry to identify congestion and advise routes.

    Accessibility agent: coordinates wheelchair routing, elevators, and quiet spaces.

    Emergency agent: flags safety issues, alerts stewards, and handles SOPs.

    1. validate_input — check injection
    2. planner — agent routing
    3. knowledge_live_data — fetch RAG
    4. parallel_agents — execute LLM
    5. decision_fusion — aggregate SOP
    6. response_gen — translate

    Session memory: tracks multi-turn questions contextually.

    Output validation: guards against prompt leaks and hallucinated gates.

    Connection pool: shared HTTP resources via the Uvicorn backend.
