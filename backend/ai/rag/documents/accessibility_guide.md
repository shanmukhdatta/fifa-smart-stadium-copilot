# Accessibility Guide

Wheelchair users should enter via Gate 3, which has ramp access and elevators to all
levels. Accessible restrooms are located next to sections 104 and 220. Sensory-friendly
quiet rooms are available near Gate 3 for fans who need a low-stimulation break.
Screen-reader compatible wayfinding is available via the mobile app. Staff at every gate
are trained in basic sign language greetings and can call an accessibility coordinator
on request.
